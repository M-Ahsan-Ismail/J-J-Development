Backdate the Entries

The Backdate Entries module allows users to enter transactions on a date prior to the current date. This is useful for recording past transactions that were not entered at the time they occurred. This module helps ensure that financial records reflect the actual date of transactions, improving the accuracy and completeness of your financial data
